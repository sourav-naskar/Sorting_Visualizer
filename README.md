# Sorting_Visualizer
A Sorting Visualizer made by Javascript Language. You can visualize following Sorting Algorithms:

1)Bubble Sort


2)Selection Sort


3)Insertion Sort


4)Merge Sort


5)Heap Sort


6)Quick Sort
